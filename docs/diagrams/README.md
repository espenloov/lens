# Architecture diagrams

This folder is ready for the Excalidraw source files and their exported images.

The first canvas follows a question from the agent to PostgreSQL, ClickHouse,
Arrow, Rust, WebAssembly, and the dashboard. Save the source as
`answer_journey.excalidraw` and export it as `answer_journey.svg`.

The second canvas explains the Query Arena learning loop. Save the source as
`query_arena_learning_loop.excalidraw` and export it as
`query_arena_learning_loop.svg`.

Both matching locations are already marked in the main README.
